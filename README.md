# Декоративные деревья

Этот проект — демонстрационный витринный сайт для магазина декоративных деревьев. Он построен на базе React, TypeScript и Vite и использует Material UI, Zustand и Zod. Данные о товарах лежат локально в JSON. Отправка заявок в Telegram выполняется через serverless‑функцию.

## Быстрый старт

1. **Установка зависимостей**:

   ```bash
   npm install
   ```

2. **Запуск в разработке**:

   ```bash
   npm run dev
   ```

3. **Сборка проекта**:

   ```bash
   npm run build
   ```

4. **Просмотр собранной версии**:

   ```bash
   npm run preview
   ```

## Структура проекта

```
decorative-trees/
├── api/              # serverless функции (пример: отправка заявки)
├── src/
│   ├── app/          # глобальные провайдеры: тема MUI, роутер
│   ├── pages/        # страницы: главная, каталог, товар, корзина, чек‑аут, политика
│   ├── widgets/      # крупные блоки (каталог с фильтрами, корзина-дровер)
│   ├── features/     # бизнес‑логика (корзина, заявка)
│   ├── entities/     # сущности (товар: типы, карточки, галерея)
│   ├── shared/       # общие компоненты, утилиты, константы, хелперы
│   └── data/         # `products.json` – список товаров
├── public/           # статические ресурсы (логотип, фавиконы)
├── .env.example      # пример окружения
├── index.html        # корневой html шаблон
├── vite.config.ts    # конфигурация Vite
└── ...
```

## Serverless

Для отправки заявок используется функция `api/lead.ts`. В переменных окружения необходимо задать:

```
TELEGRAM_BOT_TOKEN=XXX
TELEGRAM_CHAT_ID=XXX
```

Функция принимает POST‑запрос с контактом клиента, списком товаров и метаданными, проверяет хонеypot, ограничивает частоту запросов и отправляет сообщение в Telegram.
